<SortingNote />

- Use `Sortable` trait.
- Define `allowedSorts()` method.

```php
use IndexZer0\EloquentFiltering\Sort\Traits\Sortable;
use IndexZer0\EloquentFiltering\Sort\Sortable\Sort;

class Product extends Model
{
    use Sortable;

    public function allowedSorts(): AllowedSortList
    {
        return Sort::only(
            Sort::field('name'),
        );
    }
}
```
