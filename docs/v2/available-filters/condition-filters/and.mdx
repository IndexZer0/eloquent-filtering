## Example

```php
$filter = [
    'type'  => '$and',
    'value' => [
        [
            'type'   => '$like',
            'target' => 'content',
            'value'  => 'is awesome',
        ],
        [
            'type'   => '$like',
            'target' => 'content',
            'value'  => 'is not boring',
        ]
    ]
];
```

```php
$sql = Comment::filter([$filter])->toRawSql();
```

### SQL

```sql
select *
  from "comments"
 where (
       ("comments"."content" LIKE '%is awesome%')
    and
       ("comments"."content" LIKE '%is not boring%')
 )
```
