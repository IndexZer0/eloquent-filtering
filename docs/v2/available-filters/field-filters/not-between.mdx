```php
Filter::field('age', [FilterType::NOT_BETWEEN]);
```

## Example

```php
$filter = [
    'type'   => '$notBetween',
    'target' => 'age',
    'value'  => [18, 65,],
];
```

```php
$sql = Person::filter([$filter])->toRawSql();
```

### SQL

```sql
select * from "people" where "people"."age" not between 18 and 65
```
