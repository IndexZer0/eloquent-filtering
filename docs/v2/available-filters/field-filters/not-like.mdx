## Definition
```php
Filter::field('description', [FilterType::NOT_LIKE])
```

## Example

```php
$filter = [
    'type'   => '$notLike',
    'target' => 'description',
    'value'  => 'Symfony',
];
```

```php
$sql = Project::filter([$filter])->toRawSql();
```

### SQL

```sql
select * from "projects" where "projects"."description" NOT LIKE '%Symfony%'
```
