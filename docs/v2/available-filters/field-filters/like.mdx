## Definition

```php
Filter::field('description', [FilterType::LIKE])
```

## Example

```php
$filter = [
    'type'   => '$like',
    'target' => 'description',
    'value'  => 'Laravel',
];
```

```php
$sql = Project::filter([$filter])->toRawSql();
```

```sql
select * from "projects" where "projects"."description" LIKE '%Laravel%'
```
