## Definition
```php
Filter::field('price', [FilterType::BETWEEN_COLUMNS]);
```

## Example

```php
$filter = [
    'type'   => '$betweenColumns',
    'target' => 'price',
    'value'  => [
        'min_allowed_price',
        'max_allowed_price',
    ],
];
```

```php
$sql = Product::filter([$filter])->toRawSql();
```

### SQL

```sql
select
  *
from
  "products"
where
  "products"."price" between "products"."min_allowed_price"
  and "products"."max_allowed_price"
```
