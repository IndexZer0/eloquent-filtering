## Definition
```php
Filter::field('name', [FilterType::IN]);
```

## Example

```php
$filter = [
    'type'   => '$in',
    'target' => 'name',
    'value'  => ['Taylor', 'Otwell',]
];
```

```php
$sql = Person::filter([$filter])->toRawSql();
```

### SQL

```sql
select * from "people" where "people"."name" in ('Taylor', 'Otwell')
```
