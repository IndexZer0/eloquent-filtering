<Note>
<strong>Requirements</strong>
`PHP Version >= 8.2`
`&&`
`Laravel Version >= 10`
</Note>

<Steps>
  <Step title="Composer Require">
    ```bash
    composer require indexzer0/eloquent-filtering
    ```
  </Step>
  <Step title="Install">
    ```bash
    php artisan eloquent-filtering:install
    ```
  </Step>
  <Step title="Make Model Filterable">
    [Make Model Filterable](/v2/basics/make-model-filterable)
  </Step>
</Steps>
